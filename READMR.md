# 🤖 Telegram Echo Bot

A simple Telegram echo bot built with `python-telegram-bot` and deployed on Railway.

## 🚀 Features

- **Echo Messages**: Send any text and the bot will echo it back
- **Command Support**: Responds to `/start`, `/help`, `/about`, and `/echo`
- **User Friendly**: Shows user information in responses
- **Error Handling**: Graceful error management

## 🛠️ Commands

| Command | Description |
|---------|-------------|
| `/start` | Start the bot and see welcome message |
| `/help` | Show available commands |
| `/echo <text>` | Echo your text back |
| `/about` | Information about the bot |

## 📋 Prerequisites

- Python 3.11 or higher
- A Telegram Bot Token from [@BotFather](https://t.me/botfather)
- Railway account (for deployment)
